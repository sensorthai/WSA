Key Features

เทคโนโลยีที่ออกแบบมาเพื่ออาคารของคุณ

Wireless Installation

ติดตั้งง่าย ระบบไร้สาย 100% ไม่ต้องเจาะผนังหรือเดินสายไฟให้กระทบความสวยงามของคอนโด

Instant Alert

แจ้งเตือนทันทีภายใน 5 วินาที เมื่อตรวจพบน้ำนองเพียง 0.5 มม. ช่วยให้ทีมช่างเข้าระงับเหตุได้ทัน

Scalable System

เริ่มต้นจากจุดเสี่ยง (หน้าลิฟต์) และสามารถขยายเพิ่มไปยังห้องปั๊มน้ำ, สระว่ายน้ำ ได้ภายหลัง

Anti-False Alarm Design

ตัวเซ็นเซอร์ออกแบบพิเศษให้ก้านตรวจจับสามารถยกขึ้นได้ขณะแม่บ้านทำความสะอาดพื้น (ถูน้ำ) และกดลงเมื่อเสร็จงาน ป้องกันการเตือนผิดพลาด

Who is this for?

ผู้ได้รับประโยชน์จากระบบ

นิติบุคคลคอนโด

ลดความเครียดจากการจัดการปัญหาน้ำรั่วฉุกเฉินยามวิกาล ช่วยประหยัดเงินกองทุนส่วนกลางจากค่าซ่อมบำรุงลิฟต์ที่ไม่คาดคิด

Property Management

ยกระดับมาตรฐานการบริหารอาคาร (Facility Management) สร้างความได้เปรียบในการแข่งขันด้วยเทคโนโลยี IoT ทันสมัย

คณะกรรมการอาคาร

ดูแลผลประโยชน์ของลูกบ้าน ลดความเสี่ยงที่ลิฟต์จะใช้งานไม่ได้ และปกป้องมูลค่าของโครงการในระยะยาว

Simple Process

การติดตั้งและการใช้งานที่ง่ายดาย

1

Survey & Install Sensor

สำรวจหน้างาน และติดตั้งเซ็นเซอร์ไร้สายบริเวณโถงหน้าลิฟต์แต่ละชั้น (วางแนบพื้นโดยไม่ต้องเจาะ)

2

Install WaterZense Monitor

ติดตั้ง WaterZense Monitor ในห้อง Control หรือชั้นกลางๆ เพื่อเป็นศูนย์รับสัญญาณจากเซ็นเซอร์ทุกชั้น

3

Configure Fire Alarm Integration

เชื่อมต่อสัญญาณเข้ากับระบบ Fire Alarm หรือกล้องวงจรปิด (CCTV) เพื่อทริกเกอร์ระบบแจ้งเตือนส่วนกลางทันที

Monitor ผ่านระบบ Fire alarm หรือกล้องวงจรปิด

วิเคราะห์ แจ้งเตือน และตรวจสอบสถานะแบบ Real-time ได้จากระบบบริหารจัดการส่วนกลางเดิมของพื้นที่

Installation Gallery

ภาพตัวอย่างการติดตั้งจริง

โถงหน้าลิฟต์คอนโด

ติดตั้งอุปกรณ์ไร้สาย Water Zense ขนาดกะทัดรัดเพียง 10x10 ซม. โดยก้านเซ็นเซอร์ลอยเหนือพื้น 3 ซม. ดีไซน์สวยงามแนบผนัง ไม่ขวางทางเดิน

ห้องช่าง / ห้องควบคุม

ติดตั้งอุปกรณ์ WaterZense Monitor รับสัญญาณและกระจายข้อมูลเข้าระบบอาคาร หรือส่งขึ้นระบบ Cloud

ตู้แจ้งเตือน (Monitor & Siren)

ทำงานร่วมกับกล้องวงจรปิด CCTV และมีไฟไซเรนแจ้งเตือนทันทีเมื่อเกิดเหตุ เพื่อให้ช่างประจำอาคารเข้าแก้ไขสถานการณ์ได้อย่างรวดเร็ว

Premium Maintenance

อุ่นใจด้วยบริการหลังการขาย และการบำรุงรักษา

ทีมผู้เชี่ยวชาญของเราจะเข้า ตรวจสอบระบบอย่างสม่ำเสมอ (Preventive Maintenance) เพื่อเทสการเชื่อมต่อสัญญาณอุปกรณ์ และเน้นย้ำ ตรวจเช็คระดับ Battery ในตัว WaterZense Sensor ทุกตัวในอาคาร เพื่อรับประกันว่าระบบจะทำงานอย่างราบรื่น และพร้อมปกป้องคอนโดของคุณให้ปลอดภัยจากน้ำรั่วซึมอยู่เสมอตลอด 24 ชั่วโมง
